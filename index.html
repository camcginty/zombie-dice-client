Zombie Dice

Eat Brains. Don't get shot.

A game for 2 players (currently).

Player One has 0 brains.
Player Two has 0 brains.

0 Brains. You need 13 Brains total to win.

0 Shots. If you are shot 3 times this turn, you lose any brains you acquired this turn.

0 Feet. These people got away from you. If you roll again, you will reroll these dice.

Would you like to roll again and keep accumulating brains (and shots)?

Or would you like to bank your brains and let the next player go?
